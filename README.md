# LuckyFortuneRealm.com

- **Domain:** https://luckyfortunerealm.com
- **Theme:** Primary #287bff with accent #ff8c42.
- **Google Font:** 'Poppins'
- **Pages:** 12 HTML documents + robots.txt + sitemap.xml.
- **Games included:** Ace of Spades, Agent Destiny, Agent of Hearts, Alice Cooper and the Tome of Madness, Animal Madness, Ankh of Anubis
- **Highlights:** Lazy-loaded WebP art, inline critical CSS, Open Graph tags, JSON-LD WebSite+ItemList, responsive nav, internal linking, and virtual-credit messaging throughout.

## Run locally

```
npx serve .
```

## Deployment
- Upload `/images` with lossless compression preserved.
- Update canonical URLs if domain changes.
- Submit sitemap to Search Console post-launch.
